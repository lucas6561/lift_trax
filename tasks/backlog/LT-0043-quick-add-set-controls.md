# LT-0043: Add quick-add set controls

## Why

Logging common sets should be fast during a workout. Repeated typing makes mobile and high-volume training sessions feel slower than they need to.

## Outcome

The logging flow offers quick-add controls for common set, rep, and weight patterns while still allowing manual entry.

## Scope

- In scope: common quick-add buttons, repeat-last-set behavior, clear manual override, and mobile-friendly layout.
- Out of scope: automatic plate math, voice input, or replacing the full edit form.

## Acceptance criteria

- [ ] Users can repeat the previous set with one action.
- [ ] Common rep counts or set templates are available without crowding the UI.
- [ ] Manual entry remains available for unusual sets.
- [ ] Tests cover quick-add behavior and normal manual entry.
- [ ] `qualityGate` passes.

## Notes

This should be designed for tired hands and small screens.
