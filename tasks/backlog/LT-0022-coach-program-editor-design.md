# LT-0022: Design the coach program editor

## Why

Coaches should eventually be able to create multi-week programs without editing raw files.

## Outcome

A product and technical design describes a drag-and-drop editor for creating and editing program schema files.

## Scope

- In scope: workflows, screen inventory, editable schema concepts, validation feedback, import/export, and follow-up implementation tasks.
- Out of scope: full editor implementation.

## Acceptance criteria

- [ ] The design covers creating weeks, days, blocks, lift slots, exercise pools, and progression rules.
- [ ] The design explains validation feedback.
- [ ] The design supports import and export of schema files.
- [ ] Follow-up build tasks are created.

## Notes

Do not build this before the schema contract stabilizes.
