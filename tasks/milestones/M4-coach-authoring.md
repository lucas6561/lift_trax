# M4: Coach authoring

## Goal

Make program creation practical for a coach through files first, then a drag-and-drop editing experience.

## Exit criteria

- Coaches can create or import valid program schema files.
- The app reports schema problems clearly.
- Existing generators can produce editable schema-backed programs.
- The editor design is ready to implement.

## Tasks

- `LT-0019`: Build a schema validation layer.
- `LT-0020`: Create a generic schema-to-wave builder.
- `LT-0021`: Convert existing conjugate and hypertrophy builders to schema output.
- `LT-0022`: Design the coach program editor.
